# nextabase

Next.js 13 + Supabase + shadcn/ui starter template

## Features

- Next.js 13 App Directory
- shadcn/ui for component styling

## Usage

```bash
yarn
yarn supabase start
```

```bash
yarn supabase start
yarn dev
```
